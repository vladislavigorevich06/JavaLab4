## **Задание 1. Обобщенные типы (Box и Storage)**

### **1.1 Box**

* Обобщенная коробка, способная хранить один объект любого типа.
* Методы:

  * `setItem(T item)` — помещает объект в коробку (если она пуста).
  * `getItem()` — извлекает объект и обнуляет ссылку.
  * `isFull()` — проверка на заполненность.
* Демонстрация:

  * Создание коробки для числа `3`, извлечение и вывод значения.

### **1.2 Storage**

* Хранилище с альтернативным значением, если объект равен `null`.
* Методы:

  * `getItem()` — возвращает объект или альтернативу.
* Демонстрация:

  * Хранилище чисел с `null` → альтернатива `0`.
  * Хранилище чисел с `99` → альтернатива `-1`.
  * Хранилище строк с `null` → альтернатива `"default"`.
  * Хранилище строк с `"hello"` → альтернатива `"hello world"`.



## **Задание 2. Поиск максимума в коробках**

* Метод `MaxFinder.findMax(List<Box<? extends Number>> boxes)` возвращает максимальное значение среди коробок.
* Демонстрация:

  * Создание нескольких коробок с числами (`Integer`, `Double`, `Float`).
  * Вычисление максимума и вывод результата.



## **Задание 3. Обобщенные методы (GenericMethods)**

### **3.1 Map**

* Применение функции к каждому элементу списка и получение нового списка.
* Примеры:

  * Строки → длины строк.
  * Числа → абсолютные значения.
  * Массивы чисел → максимальные значения каждого массива.

### **3.2 Filter**

* Фильтрация списка по условию (Predicate).
* Примеры:

  * Строки длиной ≥ 3.
  * Неположительные числа.
  * Массивы без положительных чисел.

### **3.3 Reduce**

* Сведение списка к одному значению с использованием BinaryOperator.
* Примеры:

  * Конкатенация строк.
  * Сумма чисел.
  * Подсчет общего количества элементов в списках списков.

### **3.4 Collect**

* Преобразование списка в другую коллекцию.
* Примеры:

  * Разделение чисел на положительные и отрицательные.
  * Группировка строк по длине.
  * Удаление дубликатов.
